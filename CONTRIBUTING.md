# Contributing to CineForge AI Skills

Thank you for your interest! As a community-driven repository, we welcome new skills and agent adapters.

## Workflow
1. Create an issue to discuss your proposed skill.
2. Ensure you follow the canonical YAML frontmatter format in `SKILL.md`.
3. Provide a working example in `examples/`.
4. Open a Pull Request.

All contributions must abide by the `CODE_OF_CONDUCT.md`.
