# Security Policy

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please do not open a public issue for a security vulnerability. 
Instead, report vulnerabilities privately by emailing the maintainer or opening a GitHub private vulnerability report.
We will respond within 48 hours and work with you to patch and disclose the issue safely.
